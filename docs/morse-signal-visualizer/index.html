MORSE SIGNAL VISUALIZER

点・線・間隔で読む通信。

モールス信号は、短い点、長い線、そして無音の間隔で文字を表します。このデモでは、文字がどのように信号へ分解され、どの間隔で読まれるのかを可視化します。

ENCODED 入力した文字をモールス信号に分解します
・
短点
短く光る・短く鳴る
ー
長点
短点の約3倍の長さ
間
区切り
文字や単語を分ける無音
SIGNAL OUTPUT
現在の読み取り: 待機中
READING RULE
1. 点と線の組み合わせを1文字として読む。
2. 文字と文字の間には少し長い無音が入る。
3. 単語と単語の間にはさらに長い無音が入る。
SIGNAL STRUCTURE

文字ごとの信号と読み取り順

横に長いタイムラインではなく、1文字ごとに「文字 → モールス → 時間の長さ」を並べて表示します。スマホでは上から順に読むだけで、文字間・単語間も追える構成です。
