JavaScriptを禁止するとコンテンツを隠すページはクライアントのjavascriptで生成していると考えられます。
右クリックはできますので、ソースを表示して内容を見る事ができます（要HTMLの知識）

JavaScriptをonにしないと表示されない内容です。
右クリック禁止スクリプト（document.oncontextmenu = () => false;）が動くので制御ができません。
これを解消したい場合は、手間ですがアドレスバーにjavascript:void(document.oncontextmenu=null);を打ち込みましょう。コピペするとjavascript:が消えるので気をつけて！

ドラッグはCSS(styleタグ)のuser-selectがやっているのでスクリプトを禁止にしたぐらいでは解決しません。
開発者ツールでドラッグしたい項目のuser-selectを消すなどの対応が必要です。

JavaScript自体をOFFにすると、noscriptタグが実行されるためこの内容が表示されません。
