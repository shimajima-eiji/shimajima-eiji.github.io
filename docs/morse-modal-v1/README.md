# モールス信号ビジュアライザー モーダル組み込み版 v1

## 確認方法

1. ZIPを展開する
2. index.html をブラウザで開く
3. 再生ボタンを押す
4. ひよこ通信係の画像が、待機 / 短点 / 長点 / 完了で切り替わることを確認する
5. 「通信係の説明を開く」でモーダルを開く

## この版で確認すること

- 画像ファイル分離版で表示できるか
- モーダル上でも同じ画像を参照できるか
- 状態差分がUIの邪魔をしていないか
- ひよこ通信係が主役になりすぎていないか
- モールス信号の読み解き補助として機能しているか

## 実装方針

- Canvas確認ではなく、ローカルHTML確認を正とする
- 画像は assets/chick/*.jpg を相対パス参照
- Reactへ戻す時も同じ状態IDを使う
- 画像はCSS/SVGで再現せず、ファイルを状態で切り替える
