読み解きビュー

モールス信号を、短点と長点に分けて見る

入力した文字を、短く光る「短点」と長く光る「長点」に分けて表示します。丸暗記ではなく、いま読んでいる位置・長さ・区切りが見える画面です。

信号分解ビュー

短点 長点 左から順に読む
再生すると、現在読んでいる短点・長点が光ります。
待機

短点を見る

短点は短く光る信号です。この画面では小さなバーで表示します。

長点を見る

長点は短点より長く続く信号です。この画面では横に長いバーで表示します。

文字ごとに区切る

同じ記号列でも、区切りがあることで文字として読めます。文字カード単位でまとまりを見ます。

補足

いまの信号の読み方

短点と長点は、音や光の長さの違いとして読みます。この画面では、短点を短いバー、長点を長いバーとして表示しています。

短点短く光る。画面では短いバーとして表示。
長点短点より長く光る。画面では長いバーとして表示。
区切り文字ごとにカードを分けることで、どこまでが一文字かを見えるようにする。
