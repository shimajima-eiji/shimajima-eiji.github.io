# モールス信号の読み解き v2

## 目的

過去方針に戻したデザインに、ひよこ通信係の画像差分とモーダルを組み込んだローカル確認版です。

## 前回v1から戻した点

- 英語見出しを避け、日本語中心に変更
- 画面の主役を「モールス信号の読み解き」に戻した
- ひよこ通信係は補助役として小さめに配置
- タイムラインを別セクションにせず、文字ごとの信号分解ビューへ統合
- モーダルはキャラ説明ではなく「信号の読み方」補足に変更
- スマホで横スクロールしにくい構成に変更

## 確認方法

1. ZIPを展開
2. index.html をブラウザで開く
3. 再生する を押す
4. 短点・長点に合わせてバーとひよこが反応することを確認
5. 信号の読み方を見る を押してモーダルを確認

## 組み込み方針

画像は assets/chick/*.jpg を相対パスで参照しています。
Reactに戻す場合も、状態ID idle / dot / dash / done を維持します。
